How the Web Works

You open your computer to view the internet.
The internet is a collection of linked documents (mostly html) saved on various servers.
Servers are large computers used for storage, not for display, like your personal computers.
Your browser uses the protocal http to access all the different servers.

you > your computer/browser > the internet > http > servers

you < your computer/browser < the internet < http < servers

What is a website

A bunch of text documents read by browsers which browsers interpret the text into a display.

What coding looks like

inline vs block

block - invisibile boxes separating
< p >, < div >, < form >

inline - in the line of text
< span >, < br >, < em >

normal vs void

normal has opening and closing
< p > < / p >

void has only one set
< img src = "" >
